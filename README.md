## 🚀 Installation locale

1. **Cloner le repository**
   ```bash
   git clone https://github.com/votre-username/domaine-du-lendemain.git
   cd domaine-du-lendemain
   ```

2. **Lancer un serveur local**
   ```bash
   # Python 3
   python3 -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js
   npx serve .
   ```

3. **Ouvrir dans le navigateur**
   ```
   http://localhost:8000
   ```
